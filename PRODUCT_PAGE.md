# breakthrough player+

A premium desktop media player for Windows focused on speed, control, and clean audio.

## Hero Copy

Play everything. Control everything.
From music sessions to full video playback, breakthrough player+ gives you a modern UI, realtime EQ, spectrum visuals, subtitle support, and compact mini-player mode.

## Key Benefits

- Smooth audio/video playback for local files
- Realtime EQ with presets
- Subtitle support (`.srt` / `.vtt`) in Pro
- Playlist with right-click actions (play, rename, move, remove)
- Resume playback memory per file
- Compact Mini Player mode (always on top)
- Dark and Light themes

In app:

1. Click `Activate Pro`
2. Copy device HWID
3. Generate key for that HWID
4. Paste key
5. Click `Activate`
